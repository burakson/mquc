# Mquc (Mes que un club)
A dummy app of the [best team in the world](http://en.wikipedia.org/wiki/FC_Barcelona).

## Stack
  - Sinatra
  - Backbone.js (+Marionette.js)
  - Browserify
  - Gulp
  - Jade

## Installation
```
$ gem install bundler
$ bundle install
$ rake db:migrate && rake db:seed
$ npm install -g browserify
$ npm install -g gulp
$ npm install
$ gulp
```

**Bam!**
